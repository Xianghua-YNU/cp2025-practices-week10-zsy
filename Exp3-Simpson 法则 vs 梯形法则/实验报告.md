## 二、实验原理

### 2.1 Simpson 法则

简述 Simpson 法则的积分思想、公式推导及适用条件（$N$ 必须为偶数），并给出公式：
$$
\int_a^b f(x)dx \approx \frac{h}{3}\left[f(x_0) + 4\sum_{i=1,3,5,...}^{N-1}f(x_i) + 2\sum_{i=2,4,6,...}^{N-2}f(x_i) + f(x_N)\right]
$$

### 2.2 梯形法则

简述梯形法则的积分思想、公式推导，并给出公式：
$$
\int_a^b f(x)dx \approx h\left[\frac{f(x_0)}{2} + f(x_1) + \cdots + f(x_{N-1}) + \frac{f(x_N)}{2}\right]
$$

import numpy as np

def f(x):
    return x**4 - 2*x + 1

def trapezoidal(f, a, b, N):
  
    h = (b - a) / N
    x = np.linspace(a, b, N+1)
    y = f(x)
    integral = (h / 2) * (y[0] + 2 * np.sum(y[1:N]) + y[N])
    return integral

# Simpson法则积分函数（学生需完成）
def simpson(f, a, b, N):
 
    if N % 2 != 0:
        raise ValueError("Simpson 法则要求 N 必须为偶数")
    h = (b - a) / N
    x = np.linspace(a, b, N+1)
    y = f(x)
    integral = (h / 3) * (y[0] + 4 * np.sum(y[1:N:2]) + 2 * np.sum(y[2:N:2]) + y[N])
    return integral

def main():

    exact_integral = 4.4

    simpson_results = []

### 4.2 误差分析
- 分析两种方法在不同 $N$ 下的误差表现。
  A:
梯形法则：
当 N=100 时，梯形法则的相对误差为 1.2×10^−4。当 N=1000 时，梯形法则的相对误差为 1.2×10^−6。
Simpson 法则：
当 N=100 时，Simpson 法则的相对误差为 3.0×10^−8。当 N=1000 时，Simpson 法则的相对误差为 3.0×10^−10。
从这些结果可以看出，Simpson 法则在相同的子区间数 N 下，误差远小于梯形法则。
- Simpson 法则与梯形法则误差的数量级差异及原因。
  A:误差数量级差异：
梯形法则：误差约为 O(1/N^2)。Simpson 法则：误差约为 O(1/N^4)。在本实验中，当 N=100 时，梯形法则的误差为 1.2×10^−4，而 Simpson 法则的误差为 3.0×10^−8 ，误差差异约为 10^4倍。当 N=1000 时，梯形法则的误差为 1.2×10^−6，而 Simpson 法则的误差为 3.0×10^−10，误差差异约为 10^4倍。
原因：
梯形法则：通过将积分区间近似为梯形来计算面积，其误差主要来源于将曲线近似为直线。随着 N 的增加，梯形的数量增加，误差逐渐减小，但减小的速度为 O(1/N^2 )。Simpson 法则：通过将积分区间近似为抛物线来计算面积，其误差来源于将曲线近似为二次多项式。由于二次多项式可以更好地拟合曲线，因此Simpson 法则的误差减小速度更快，为 O(1/N^4)。
- 讨论 $N$ 增大时误差的变化趋势。
  A:梯形法则：当 N 增大时，梯形法则的误差按 O(1/N^2) 的速度减小。当 N=100 时，误差为 1.2×10^−4；当 N=1000 时，误差为 1.2×10^−6，误差减小了约 100 倍（与 N^2成反比）。Simpson 法则：当 N 增大时，Simpson 法则的误差按 O(1/N^4) 的速度减小。当 N=100 时，误差为 3.0×10^−8；当 N=1000 时，误差为 3.0×10^−10，误差减小了约 10,000 倍（与 N^4成反比）。

## 五、结论
- 总结两种方法的适用场景和优劣。
  A:梯形法则：优点：实现简单，计算量较小。对精度要求不高的情况。缺点：误差较大，尤其是对于复杂函数，需要较大的 N 才能达到较高的精度。
Simpson 法则：优点：误差小，精度高，适合对精度要求较高的情况。缺点：实现相对复杂，需要确保 N 为偶数。
- 结合本实验结果，说明实际工程中如何选择数值积分方法。
  A:如果对精度要求较高，尤其是需要快速减小误差的场景，选择 Simpson 法则。如果对精度要求较低，或者计算资源有限，选择梯形法。对于光滑函数（如多项式函数），Simpson 法则通常表现更好。对于非光滑函数或存在间断点的函数，梯形法则可能更稳定。
